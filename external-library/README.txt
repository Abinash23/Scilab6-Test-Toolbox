This directory is a library source for a transpose function defined in C. 

1. A simple "trans.c" is wrriten in C to calulate tranpose of a matrix.
2. The "run.sh" shell script can be used to compile this C program into a shared object file (.so file) on a Linux system. On Windows OS, follow the instructions given inside the windows-compile.txt file.
3. This directory also contains a compiled and ready-to-use library inside the "thirdparty" directory.
4. Now copy the "thirdparty" directory to the test toolbox directory.
